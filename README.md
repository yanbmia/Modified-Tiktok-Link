# Modified-Tiktok-Link

I deleted Tiktok, but my friends keep sending me Tiktoks. So I made this! 

Open the tiktok link your friend sends you, then copy the link from the search engine. Note: this doesn't work if you directly copy the link.

Happy viewing!

# Access via Web

https://yanbmia.github.io/Modified-Tiktok-Link/


## Alternative: Installation

Install Flask:
```bash
pip install flask
```
Run the app:
```bash
python app.py
```
Open your browser and go to http://127.0.0.1:5000 to access the URL Modifier web interface.
